Clear button clear the shape in the canvas.

Save project save the whole project in json format.

Load project load the save project into the canvas.

Export button export the project into video format. It use browser video tag.

There is already example project saved for you in the project directory. You can load it using Load project button.

In case you cannot draw any shape, try double clicking on the canvas to unselected the selected shape. It is explicitly coded to not draw the shapes while there is one shape selected.
